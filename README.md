Full-Stack Task Management System
A complete MERN-based task management application with authentication, real-time updates, notifications, WebSockets, GraphQL API, and an enhanced React frontend.

Features
Authentication
 User signup & login
 Password hashing (bcryptjs)
 JWT-based authentication
 HTTP-only secure cookies

Task Management (REST API)
 Create, Read, Update, Delete tasks
 User-specific tasks
 Role-based access (admin/user)
 Input validation & error handling

Notification System
 Task-related notifications
 Mark as read / mark all as read
 Delete notifications
 Unread filters

Real-Time Features (Socket.io)
 Real-time task updates (create/update/delete)
 Real-time chat system
 Typing indicators
 User join/leave events
 Real-time notifications

GraphQL API
 GraphQL queries for tasks, notifications, and messages
 GraphQL mutations for creating & updating tasks
 Apollo Server integration

Frontend (React)
 Modern UI
 Login / Signup pages
 Dashboard with real-time updates
 Notifications panel
 Chat system
 Enhanced pages with updated UX

Tech Stack
  Backend
    Node.js, Express.js
    MongoDB, Mongoose
    JWT Authentication
    Socket.io
    Apollo GraphQL Server

  Frontend
    React.js (with Hooks)
    Fetch/Axios
    React Router
    Custom UI components
